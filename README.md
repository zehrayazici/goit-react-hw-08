# goit-react-hw-08

A full-featured contact management app with user authentication, built with React, Redux Toolkit, and React Router.

✨Features

User registration & login with JWT authentication
Persistent session — token stored in localStorage via redux-persist
Private routes — only logged-in users can access contacts
Add and delete contacts
Filter contacts by name or phone number
Contacts cleared from state on logout
Form validation with Formik & Yup

⚙️Tech Stack
React 18 Vite React Router v6 Redux Toolkit ReduxReact Axios Formik Yup CSS
